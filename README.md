#  Educational Center Management System

##  About The Project
A smart desktop application built with Python to help teachers and educational centers manage their students efficiently. The system handles student registration, tracks monthly payments, and automatically exports dynamic data to Excel spreadsheets.

##  Key Features
* **Smart Search Engine:** Instantly search for students by Name or ID with real-time financial status updates.
* **Payment Tracking:** Track monthly payments across different educational levels.
* **Database Management:** Securely stores student records and payment histories using SQLite.
* **Excel Export & Auto-Save:** Automatically generates and updates formatted `.xlsx` files using `openpyxl`.
* **Interactive GUI:** User-friendly interface built with `tkinter` and `ttk`.

##  Built With
* **Python 3**
* **Tkinter** (Graphical User Interface)
* **SQLite3** (Database)
* **Openpyxl** (Excel manipulation & formatting)
